# Épico: Ajuda
### Pedido
 
### US11: Criar pedidos de ajuda com título, descrição e categoria
- Eu, como usuário, gostaria de criar [pedidos de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) com título, descrição e [categorias](/pages/modeling/lexicon?id=categoria).
 
#### Critérios de Aceitação
- Deve haver um botão que direciona para a página de criar um novo [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
- Deve haver um campo de inserção de texto para o título da [ajuda](/pages/modeling/lexicon?id=ajuda).
- Deve haver um campo de inserção de texto para a descrição da [ajuda](/pages/modeling/lexicon?id=ajuda).
- Deve haver um campo de seleção de para que o usuário selecione a [categoria](/pages/modeling/lexicon?id=categorias) da [ajuda](/pages/modeling/lexicon?id=ajuda).
- Deve haver um botão finalizar a criação do [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
 
### US12: Aparecer pedido de ajuda no mapa
- Eu, como usuário, gostaria que meu [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) aparecesse no [mapa](/pages/modeling/lexicon?id=mapa) para outros usuários.
 
#### Critérios de Aceitação
- Deve haver um [mapa](/pages/modeling/lexicon?id=mapa).
- Deve mostrar meu [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) no [mapa](/pages/modeling/lexicon?id=mapa) para outros usuários.
- Não deve mostrar meu próprio [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) no [mapa](/pages/modeling/lexicon?id=mapa) para mim.
 
### US13: Apagar meu pedido de ajuda a qualquer momento
- Eu, como usuário, gostaria de apagar meu [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) a qualquer momento.
 
#### Critérios de Aceitação
- Deve haver um botão que remove o [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
- Deve haver abrir um modal para confirmar a remoção da ajuda.
- O [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) removido não deve aparecer no [histórico](/pages/modeling/lexicon?id=histórico).
 
### US14: Escolher um possível ajudante
- Eu, como usuário, gostaria de escolher um [possível ajudante](/pages/modeling/lexicon?id=ajudante) para meu [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
 
#### Critérios de Aceitação
- Deve haver um número no [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) que informe a quantidade de [possíveis ajudantes](/pages/modeling/lexicon?id=ajudante).
- Deve haver uma lista que mostre os [possíveis ajudantes](/pages/modeling/lexicon?id=ajudante).
- Cada [possível ajudante](/pages/modeling/lexicon?id=ajudante) deve ser representado por um botão na lista.
- Assim que um [possível ajudante](/pages/modeling/lexicon?id=ajudante) for selecionado deve surgir um modal de confirmação.
 
### US15: Entrar em contato com o usuário que me ofereceu ajuda
- Eu, como usuário gostaria, de [entrar em contato](/pages/modeling/lexicon?id=abrir-serviços-externos) com o usuário que me ofereceu [ajuda](/pages/modeling/lexicon?id=ajuda).
 
#### Critérios de Aceitação
- Deve haver um botão que faz link com o [Whatsapp](/pages/modeling/lexicon?id=abrir-serviços-externos)
- Deve haver um botão que faz link com o [Google Maps](/pages/modeling/lexicon?id=abrir-serviços-externos).
- Os botões devem aparecer somente para o usuário que é [ajudante](/pages/modeling/lexicon?id=ajudante).
 
### US16: Finalizar meu pedido de ajuda assim que ele for concluído
- Eu, como usuário, gostaria de [finalizar](/pages/modeling/lexicon?id=ajudas-finalizadas) meu [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) assim que ele for concluído.
 
#### Critérios de Aceitação
- Deve haver um botão para finalizar [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) tanto para o usuário, quanto para o [ajudante](/pages/modeling/lexicon?id=ajudante).
- Quando o [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) for finalizado deve mudar seu estado para [ajuda finalizada](/pages/modeling/lexicon?id=ajudas-finalizadas).
- O [pedido de ajuda finalizado](/pages/modeling/lexicon?id=ajudas-finalizadas) deve ir para o [histórico](/pages/modeling/lexicon?id=histórico).
 
### US17: Visualizar um histórico dos meus pedidos de ajuda finalizados
- Eu, como usuário, gostaria de visualizar um [histórico](/pages/modeling/lexicon?id=histórico) dos meus [pedidos de ajuda finalizados](/pages/modeling/lexicon?id=ajudas-finalizadas).
 
#### Critérios de Aceitação
- Deve haver uma lista de [pedidos de ajuda finalizados](/pages/modeling/lexicon?id=ajudas-finalizadas).
- Cada [pedidos de ajuda finalizado](/pages/modeling/lexicon?id=ajudas-finalizadas) deve ser um botão.
- Quando apertar no botão deve exibir as informações do [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
 
### US18: Visualizar meus pedidos de ajuda abertos
- Eu, como usuário, gostaria de visualizar meus [pedidos de ajuda abertos](/pages/modeling/lexicon?id=ajudas-aberta).
 
#### Critérios de Aceitação
- Deve haver uma lista com os [pedidos de ajuda abertos](/pages/modeling/lexicon?id=ajudas-aberta).
- Cada [pedidos de ajuda abertos](/pages/modeling/lexicon?id=ajudas-aberta) deve ser um botão.
- Quando apertar no botão deve exibir as informações do [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
- Deve haver um número no [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) que informe a quantidade de [possíveis ajudantes](/pages/modeling/lexicon?id=ajudante).
- Deve haver um botão que remove o [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
## Oferta
 
### US19: Ofertar ajuda à pedidos que estão abertos no mapa
- Eu, como usuário, gostaria de [ofertar ajuda](/pages/modeling/lexicon?id=oferta) ajuda à [pedidos que estão abertos](/pages/modeling/lexicon?id=ajudas-em-andamento) no [mapa](/pages/modeling/lexicon?id=mapa).
 
#### Critérios de Aceitação
- Deve haver balões no [mapa](/pages/modeling/lexicon?id=mapa) que representam os [pedidos de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
- Ao apertar no balão deve mostrar informações básicas do [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
- Ao apertar no balão duas vezes deve mostrar as informações básicas do [pedidos de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) e a possibilidade de [ofertar ajuda](/pages/modeling/lexicon?id=oferta).
 
### US20: Poder finalizar minha oferta de ajuda após ajudar outro usuário
- Eu, como usuário, gostaria de poder finalizar minha [oferta de ajuda](/pages/modeling/lexicon?id=oferta) após ajudar outro usuário.
 
#### Critérios de Aceitação
- Deve haver um botão para finalizar [oferta de ajuda](/pages/modeling/lexicon?id=oferta).
- Quando o [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) for finalizado deve mudar seu estado para [ajuda finalizada](/pages/modeling/lexicon?id=ajudas-finalizadas).
- A [oferta de ajuda finalizada](/pages/modeling/lexicon?id=ajudas-finalizadas) deve ir para o [histórico](/pages/modeling/lexicon?id=histórico).
 
### US21: Visualizar minhas ofertas de ajuda em andamento
- Eu, como usuário, gostaria de visualizar minhas [oferta](/pages/modeling/lexicon?id=oferta) de [ajuda em andamento](/pages/modeling/lexicon?id=ajudas-em-andamento).
 
#### Critérios de Aceitação
- Deve haver uma lista com as ofertas dos [pedidos de ajuda abertos](/pages/modeling/lexicon?id=ajudas-em-andamento).
- Cada oferta de [pedido de ajuda aberto](/pages/modeling/lexicon?id=ajudas-em-andamento) deve ser um botão.
- Quando apertar no botão deve exibir as informações do [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
- Deve haver botões para que o [ajudante](/pages/modeling/lexicon?id=ajudante) [entre em contato](/pages/modeling/lexicon?id=abrir-serviços-externos) com quem pediu ajuda.
 
### US22: Ser notificado quando tiverem aceito uma oferta de ajuda minha
- Eu, como usuário, gostaria de ser notificado quando tiverem aceito uma [oferta de ajuda](/pages/modeling/lexicon?id=oferta) minha.
 
#### Critérios de Aceitação
- Deve haver uma página com as notificações.
- Quando um [ajudante](/pages/modeling/lexicon?id=ajudante) for escolhido para a ajuda deve informar o usuário.
- A notificação deve ser curtas, diretas e com informações relevantes para o usuário.
 
### US23: Ter acesso à rota até o pedido de ajuda
- Eu, como usuário, gostaria de ter acesso a rota até o [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
 
#### Critérios de Aceitação
- Deve haver um botão especial, na página de informação da ajuda, após o [ajudante](/pages/modeling/lexicon?id=ajudante) ter sido escolhido.
- O botão do deve redirecionar o usuário para o [Google Maps](/pages/modeling/lexicon?id=abrir-serviços-externos).
- Não é necessário a aparição de um modal de confirmação.
 
### US24: Entrar em contato com o usuário que vou ajudar
- Eu, como usuário, gostaria de entrar em contato com o usuário que vou [ajudar](/pages/modeling/lexicon?id=ajuda).
 
#### Critérios de Aceitação
- Deve haver um botão especial, na página de informação da ajuda, após o [ajudante](/pages/modeling/lexicon?id=ajudante) ter sido escolhido.
- O botão do deve redirecionar o usuário para o [Whatsapp](/pages/modeling/lexicon?id=abrir-serviços-externos).
- Não é necessário a aparição de um modal de confirmação.
 
### US25: Fazer diversas ofertas de ajuda
- Eu, como usuário, gostaria de fazer diversas [ofertas de ajuda](/pages/modeling/lexicon?id=oferta).
 
#### Critérios de Aceitação
- Deve haver uma lista com as [ofertas](/pages/modeling/lexicon?id=oferta) de ajuda do usuário.
- Cada [oferta](/pages/modeling/lexicon?id=oferta) deve ser um botão
- Ao apertar em uma oferta deve informar os detalhes da mesma, mostrar botões do [Whatsapp e do Google Maps](/pages/modeling/lexicon?id=abrir-serviços-externos) e também um botão para finalizar a mesma.
 
### US26: Visualizar um histórico das minhas ofertas de ajuda finalizadas
- Eu, como usuário, gostaria de visualizar um [histórico](/pages/modeling/lexicon?id=histórico) das minhas [ofertas](/pages/modeling/lexicon?id=oferta) de [ajuda finalizadas](/pages/modeling/lexicon?id=ajudas-finalizadas).
 
#### Critérios de Aceitação
- Deve haver uma lista de [ofertas de ajuda finalizadas](/pages/modeling/lexicon?id=ajudas-finalizadas).
- Cada [ofertas de ajuda finalizadas](/pages/modeling/lexicon?id=ajudas-finalizadas) deve ser um botão.
- Quando apertar no botão deve exibir as informações do[categorias](/pages/modeling/lexicon?id=categoria).
 
### US27: Filtrar os pedidos de ajuda antes de fazer uma oferta
- Eu, como usuário, gostaria de [filtrar os pedidos de ajuda](/pages/modeling/lexicon?id=filtrar-pedidos-de-ajuda) antes de fazer uma [oferta](/pages/modeling/lexicon?id=oferta) para que encontre [categorias](/pages/modeling/lexicon?id=categorias) específicas de [ajudas](/pages/modeling/lexicon?id=ajuda).
 
#### Critérios de Aceitação
- Deve haver um botão, no [mapa](/pages/modeling/lexicon?id=mapa), para escolher a [categoria](/pages/modeling/lexicon?id=categoria) do [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda).
- Ao apertar o botão deve-se abrir um modal, no qual o usuário pode escolher 1 dentre as diversas [categorias](/pages/modeling/lexicon?id=categoria).
- Ao escolher uma [categoria](/pages/modeling/lexicon?id=categoria), o [mapa](/pages/modeling/lexicon?id=mapa) deve retornar somente os [pedido de ajuda](/pages/modeling/lexicon?id=pedido-de-ajuda) com [categoria](/pages/modeling/lexicon?id=categoria) idêntica.
